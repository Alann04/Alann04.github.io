Slime's Journey


Slime's Journey Game

About Slime's Journey:

Story

The game is about a slime that runs into a swarm of bees, and is
trying to dodge them as long as possible

Code

At first I wanted to implement a scoring system based on coins but
some dreams don't become reality, so I based my scoring on the time.
Every second, the player would obtain a point. The RNG that is implemented
in the game is used for the cloud positioning and the bee's positioning.
The difficulty will change every 15 seconds The player will lose when the
bees touch the slime, there is no other way for the game to start.

Behind the Scenes

My game was started with a single sprite, the bee. I really wanted one
of my sprites to be based on another character from another popular pixelated
series. The slime was implemented in the later stages because I needed a
character that is known to jump. When making the game, I was particularly
happy with coding the death circumstances for the slime, because that felt
like my first step towards real progress. Another point is the tutorial screen,
which features the bee showing off a grand dialogue. The biggest obstacle
was getting the collision for the bee and slime, and the slime and floor.
Each sprite messed with the others. Another issue was getting the bees to
create more bees that would do the same thing. However I did not add a crucial
component in the early stages, and a prominent glitch I called "the swarm"
would get me back into the blocks.

Future Ideas

The product, Slime's Journey, will need a lot more development before
I decide that it is ready, one element I want to add is for bees to come
from every edge of the screen to give the player a more difficult experience.
I would also add more duplicate clouds to make the task of dodging bees
easier for players, however, I am quite satisfied with the product so far

Favorite Script

My favorite script to code of the game was the cloning of the bee. I am
particularly proud of that piece of code because it took a while to perfect.
the more blocks were also extremely fun to work with as well.
